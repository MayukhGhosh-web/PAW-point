# PAW-point

PAW-point is a veterinary appointment and pet healthcare platform built for pet owners and veterinarians to simplify appointment booking, pet management, and clinic workflows.

## Tech Stack

* React
* TypeScript
* TanStack Start
* Tailwind CSS
* shadcn/ui
* Framer Motion
* Vite

---

## Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/PAW-point.git
```

Install dependencies:

```bash
npm install --legacy-peer-deps
```

Start the development server :

```bash
npm run dev
```

---

## Branch Workflow

| Branch            | Purpose                      |
| ----------------- | ---------------------------- |
| main              | Stable production-ready code |
| develop           | Integration and testing      |
| personal branches | Individual contributor work  |

### Contribution Rules

* Work only on your assigned branch
* Create Pull Requests into `develop`
* Do not push directly to `main` or `develop`



Push changes:

